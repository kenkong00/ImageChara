# AI角色编辑器 (AI Character Editor)

这是一个用于构建和生成聊天AI角色性格的专业工具。

## 功能概述

### 核心属性配置
- **基础信息**: 姓名、性别、国籍、年龄
- **外貌特征**: 身高、体型、发色、瞳色、肤色、穿着风格等
- **性格特征**: 性格类型、MBTI、价值观、兴趣爱好
- **背景故事**: 出身背景、成长经历、重要事件
- **语言风格**: 说话方式、口头禅、语气特点
- **行为模式**: 社交方式、决策风格、情感表达

### 角色生成能力
1. **智能扩展**: 基于用户提供的基础信息，自动丰富角色细节
2. **性格一致性**: 确保角色的言行举止符合设定的性格特征
3. **多维度塑造**: 从心理、社交、情感等多角度完善角色

## 输出格式

生成的角色信息以JSON格式输出，兼容主流AI聊天平台：

```json
{
  "name": "角色名称",
  "description": "角色描述",
  "personality": "性格特征",
  "scenario": "场景设定",
  "first_mes": "开场白",
  "mes_example": "对话示例",
  "creator_notes": "创作者备注",
  "tags": ["标签1", "标签2"],
  "creator": "创建者",
  "character_version": "1.0",
  "create_date": "创建日期",
  "data": {
    "extensions": {},
    "character_books": []
  }
}
```

## 角色生成流程

1. **收集基础信息**: 引导用户提供角色核心属性
2. **智能扩展**: 基于输入自动生成缺失的细节
3. **一致性检查**: 确保各属性之间逻辑自洽
4. **输出成品**: 生成完整的角色配置文件

## 注意事项

- 所有内容使用中文处理和输出
- 保持角色性格的一致性和可信度
- 避免生成不当或敏感内容
- 支持角色的持续迭代和优化
